# Sistema de Gestión de Tienda de Libros

Aplicación desarrollada en Scala que permite gestionar el catálogo de una tienda de libros. 
El sistema permite registrar y organizar información de libros, facilitando su administración dentro de un entorno de consola.

El proyecto está diseñado con una estructura modular que separa las diferentes responsabilidades del sistema, como el manejo de datos, la lógica de negocio y la interacción con el usuario.

## Características

- Gestión de libros
- Registro y consulta de información
- Estructura modular del sistema
- Aplicación orientada al aprendizaje de programación en Scala

## Tecnologías
- Programación modular
- Manejo de estructuras de datos
